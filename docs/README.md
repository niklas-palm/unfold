## Examples

Study the included examples before building your own:

| Example | Description |
|---------|-------------|
| `examples/dns/` | How DNS works — 15 slides, 6 drilldowns |
| `examples/k8s/` | Kubernetes architecture — diagram-heavy |
| `examples/transformer-architecture/` | Transformer internals — educational deep-dive |

Browse examples locally:

```bash
npm run examples
```
